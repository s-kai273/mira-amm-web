## Install [direnv](https://direnv.net/docs/installation.html#from-system-packages)

<Tabs groupId="direnv-installation-commands" items={['macOS', 'Arch Linux', 'Other']} persist>
  <Tab value="macOS">
   ```sh title="Shell"
   brew install direnv
   ```
</Tab>
  <Tab value="Arch Linux">
   ```sh title="Shell"
   sudo pacman -Syu direnv
   ```
  </Tab>
  <Tab value="Other">
    See [docs](https://direnv.net/docs/installation.html#from-system-packages)
  </Tab>
</Tabs>


Then add corresponding shell hook.

<Tabs groupId="direnv-installation-commands" items={['Bash/Zsh', 'Other']} persist>
  <Tab value="Bash/Zsh">
   ```sh title="Shell"
    shell=$(basename "$SHELL") # Detect current shell
    rc_file="$HOME/.${shell}rc" # Determine corresponding config file
    hook="eval \"\$(direnv hook $shell)\"" # construct direnv hook line
    grep -qxF "$hook" "$rc_file" || echo "$hook" >> "$rc_file" # Append if not already in file
    ```
  </Tab>

## Install [devenv](https://devenv.sh)

```sh title="Shell"
nix-env --install --attr devenv -f https://github.com/NixOS/nixpkgs/tarball/nixpkgs-unstable
```

<Accordions>
    <Accordion title="I'm getting an error">
    You should see something like:

    ```
    direnv: using devenv
    • Building shell ...
    Using Cachix: devenv
    Failed to set up binary caches:

    https://devenv.cachix.org

    devenv is configured to automatically manage binary caches with `cachix.enable = true`, but cannot do so because you are not a trusted user of the Nix store.

    You have several options:

    a) To let devenv set up the caches for you, add yourself to the trusted-users list in /etc/nix/nix.conf:

        trusted-users = root yourusername

    Then restart the nix-daemon:

        $ sudo systemctl restart nix-daemon

    b) Add the missing binary caches to /etc/nix/nix.conf yourself:

        extra-substituters = https://devenv.cachix.org
        extra-trusted-public-keys = devenv.cachix.org-1:w1cLUi8dv3hnoSPGAuibQv+f9TZLr6cv/Hm9XgU50cw=

    c) Disable automatic cache management in your devenv configuration:

        {
        cachix.enable = false;
        }

    Failed to get cachix caches due to evaluation error
    ```

